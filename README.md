# Francesco Puglia

**Operations, Data & Automation builder focused on local-first systems, dashboards, workflow automation and knowledge-management tools.**

I build practical digital systems that turn scattered information, goals and workflows into structured execution.

My current portfolio is focused on four areas:

* **Operations systems** — tools for tracking work, applications, follow-ups, documents and KPIs.
* **Data & dashboards** — structured reporting, progress tracking and measurable workflows.
* **Knowledge systems** — document organisation, evidence mapping and source-first reasoning.
* **Local-first applications** — privacy-conscious tools that run locally and keep user data under control.

## Current Portfolio

### UN Knowledge & Programme Support Assistant

Evidence-based application intelligence demo for UN-style programme support and knowledge-management workflows.

The system turns a UN-style vacancy into structured requirements, maps each requirement to verifiable evidence, exposes gaps, generates checklists and produces a human-review brief.

**Core ideas:** evidence mapping, requirement extraction, local-first workflow, no-source-no-claim discipline.

**Relevant for:** Programme Support, Knowledge Management, Data/Reporting Support, Operations Support.

---

### Career Command Center

Local-first desktop dashboard for managing job applications, CV versions, public tenders, target companies, contacts, portfolio projects, follow-ups, weekly reviews and career KPIs.

**Core ideas:** job-search operations, KPI tracking, document management, follow-up discipline, local SQLite storage.

**Relevant for:** Product Operations, SaaS Support Operations, Data/BI, RevOps, Workflow Automation.

---

### Life Tracker

Goal-centric productivity system for planning, time blocking, habits, analytics and measurable execution.

The project is built around a simple principle: goals become useful only when they are translated into projects, tasks, routines, time blocks and weekly review cycles.

**Core ideas:** personal execution, goal tracking, habits, analytics dashboards, AI-assisted planning logic.

---

### Chess Training Platform

Performance-training system for chess calculation, tactics, PGN study, blindfold training and measurable improvement workflows.

**Core ideas:** structured training, spaced repetition, performance tracking, visualisation practice, domain-specific learning workflows.
